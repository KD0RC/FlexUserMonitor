# FlexUserMonitor
Utility to allow monitoring and controlling Flex radios that multiple 
people can log into.  Has a security file to allow only listed stations 
to connect.  Stations can be set to allow or disallow transmit.  Designed 
to make it easier to manage club or shared radios.
